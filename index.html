Project Details

Terms & Conditions

Purpose

This project is a personal learning assistant built for Parth. It is made for study, practice, productivity, music playback, local command launching, and understanding how frontend, backend, voice recognition, and APIs work together.

Local Backend

Computer apps and Windows features open only when the local Jarvis backend is running on localhost. The browser page alone cannot safely open system apps, so the backend uses an allowlist of approved commands.

Privacy

Voice recognition is handled by the browser after microphone permission is allowed. Chat messages may be sent to the selected AI provider only when a valid API key is configured. Personal API keys should stay inside the local .env file.

Safety

The assistant is designed to open known apps, folders, websites, and settings. It should not be used to run unknown scripts, delete files, bypass security prompts, or perform harmful system actions.

Music

Local songs placed in assets/songs are shown in the playlist. Only files that you have permission to use should be added. External music links open in a new browser tab.

Limitations

Some apps may not open if they are not installed, if Windows protocol links are disabled, or if the backend is not restarted after code changes. Chrome or Edge is recommended for voice recognition.
